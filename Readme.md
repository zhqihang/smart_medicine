# 智慧医药系统
## 1. 项目介绍
基于Springboot，搭建智慧医药系统，集成通义千问，实现智能问诊功能。

项目访问地址：[智慧医疗系统](http://106.14.46.112:8081)
## 2. 实现功能
### 2.1 系统角色
1. 游客：具备登录注册、疾病、药品的搜索、查询详情等权限
2. 用户：游客基础上增加登录、个人资料修改、登录密码修改、意见反馈、智能医生咨询等权限
3. 管理员：用户基础上增加疾病管理、药品管理、反馈管理等权限
### 2.2 后端技术栈
1. Springboot
2. SpringMVC
3. MybatisPlus
4. Thymeleaf
5. Druid
## 3. 部署步骤
1. git clone 到本地
2. 修改配置文件（数据库、邮箱等）
3. 新增阿里云OSS配置、通义千问ai-key
